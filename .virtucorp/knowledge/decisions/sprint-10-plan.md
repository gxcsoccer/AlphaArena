## 详细规划

### Issue #195: RSI 策略

**目标：** 实现相对强弱指数(RSI)交易策略

**技术要点：**
- 创建 `src/strategy/RSIStrategy.ts`
- 继承 Strategy 基类
- RSI 计算: `RSI = 100 - (100 / (1 + RS))`
- 超买(>70)卖出，超卖(<30)买入

**验收标准：**
- RSIStrategy 类实现
- 单元测试覆盖
- CLI 可使用

### Issue #196: MACD 策略

**目标：** 实现移动平均线收敛发散(MACD)交易策略

**技术要点：**
- 创建 `src/strategy/MACDStrategy.ts`
- MACD 线 = EMA(12) - EMA(26)
- 信号线 = EMA(MACD, 9)
- 金叉买入，死叉卖出

**验收标准：**
- MACDStrategy 类实现
- 单元测试覆盖
- CLI 可使用

### Issue #197: 用户偏好设置

**目标：** 保存用户的主题、语言等偏好设置

**技术要点：**
- 创建 `src/client/store/settingsStore.ts`
- 使用 localStorage 持久化
- 支持主题切换（深色/浅色）
- 支持语言切换（中文/英文）

**验收标准：**
- Store 实现
- 设置面板 UI
- 持久化功能

### Issue #198: 交易面板 UX 优化

- 实时输入验证
- Toast 错误提示
- 下单按钮 loading 状态
- 数量快捷按钮（25%/50%/75%/100%）

**验收标准：**
- 输入验证改进
- 错误提示优化
- 交互反馈提升
