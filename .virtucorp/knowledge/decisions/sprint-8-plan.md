## 详细规划

### Issue #181: 修复 Trades 页面

**问题：** 页面显示「组件加载失败」

**调查方向：**
1. 检查组件导入是否正确
2. 检查数据依赖是否满足
3. 检查 API 端点是否正常
4. 检查控制台错误日志

**修复策略：**
1. 定位根本原因
2. 修复问题
3. 添加单元测试
4. 验证修复效果

### Issue #182: 修复 Holdings 页面

**问题：** 页面显示「组件加载失败」

**调查方向：**
- 与 Trades 页面类似，检查组件、数据、API

### Issue #183: E2E 测试扩展

**测试范围：**
- 所有核心页面加载测试
