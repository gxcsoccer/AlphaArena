# 价格提醒配置教程

AlphaArena 价格提醒功能帮您实时监控市场，在价格达到目标时及时通知您。

## 功能概述

- 📈 **价格触发** - 高于/低于目标价时触发
- 🔔 **多种通知方式** - 应用内、浏览器推送、邮件、飞书
- 🔄 **循环提醒** - 持续监控，反复触发
- ⏰ **定时过期** - 设置提醒有效期

## 访问价格提醒

价格提醒功能位于交易页面的右侧面板：

![Price Alerts Panel](./images/alerts-panel.png)

## 创建提醒

### 步骤 1：打开创建面板

点击"创建提醒"按钮。

### 步骤 2：填写提醒信息

| 字段 | 说明 | 示例 |
|------|------|------|
| 交易对 | 监控的交易对 | BTC/USDT |
| 条件类型 | 高于/低于目标价 | 高于目标价 |
| 目标价格 | 触发提醒的价格 | 70000 |
| 通知方式 | 如何接收通知 | 应用内通知 |
| 提醒类型 | 单次/循环 | 循环提醒 |
| 过期时间 | 提醒有效期 | 2024-12-31 |
| 备注 | 备注信息 | 重要支撑位 |

### 步骤 3：确认创建

检查信息无误后点击"创建"按钮。

## 条件类型

### 高于目标价

当价格上涨超过目标价时触发。

**使用场景**：
- 突破阻力位时买入
- 止盈触发提醒
- 追踪止盈价位

**示例**：
```
BTC/USDT 当前价格 $65,000
设置"高于 $70,000"
当价格涨到 $70,001 时触发提醒
```

### 低于目标价

当价格下跌低于目标价时触发。

**使用场景**：
- 跌破支撑位时卖出
- 止损触发提醒
- 逢低买入机会

**示例**：
```
BTC/USDT 当前价格 $65,000
设置"低于 $60,000"
当价格跌到 $59,999 时触发提醒
```

## 通知方式

### 应用内通知

- 在应用内显示提醒消息
- 需要保持页面打开
- 点击通知可查看详情

### 浏览器推送

- 即使页面关闭也能收到通知
- 需要授权浏览器通知权限
- 首次使用会提示授权

**授权步骤**：
1. 浏览器会弹出授权请求
2. 点击"允许"启用推送
3. 如果之前拒绝，需要在浏览器设置中重新允许

### 飞书通知

- 发送到飞书机器人
- 需要配置飞书 Webhook
- 适合团队协作

### 邮件通知

- 发送邮件提醒
- 需要绑定邮箱
- 适合重要提醒

## 提醒类型

### 单次提醒

- 触发后自动禁用
- 适合一次性事件

**使用场景**：
- 止盈止损
- 特定价格位交易

### 循环提醒

- 触发后继续监控
- 价格再次达到时再次触发

**使用场景**：
- 支撑阻力位监控
- 价格区间交易

## 管理提醒

### 查看提醒列表

在价格提醒面板可以看到所有提醒：

| 标签 | 内容 |
|------|------|
| 生效中 | 当前活跃的提醒 |
| 已触发 | 已触发的历史提醒 |

### 编辑提醒

1. 点击提醒行的编辑图标
2. 修改目标价格、条件等
3. 点击"保存"

**注意**：交易对不可修改。

### 启用/禁用提醒

- 点击铃铛图标切换状态
- 禁用后不会触发通知
- 可以随时重新启用

### 删除提醒

1. 点击删除图标
2. 确认删除
3. 删除后无法恢复

## 使用示例

### 示例 1：突破交易

```
场景：BTC 突破 $70,000 阻力位后追入

创建提醒：
- 交易对：BTC/USDT
- 条件：高于目标价
- 目标价：70000
- 通知：浏览器推送
- 类型：单次

触发后：
- 收到通知
- 查看市场情况
- 决定是否入场
```

### 示例 2：止损提醒

```
场景：持有 ETH，设置止损

创建提醒：
- 交易对：ETH/USDT
- 条件：低于目标价
- 目标价：3000
- 通知：浏览器推送 + 飞书
- 类型：单次
- 备注：止损位

触发后：
- 收到通知
- 执行止损操作
```

### 示例 3：区间交易

```
场景：BTC 在 $60,000 - $70,000 区间震荡

创建提醒 1（下沿买入）：
- 交易对：BTC/USDT
- 条件：低于目标价
- 目标价：61000
- 类型：循环

创建提醒 2（上沿卖出）：
- 交易对：BTC/USDT
- 条件：高于目标价
- 目标价：69000
- 类型：循环
```

## 使用限制

### 免费版

- 最多 5 个活跃提醒
- 仅应用内通知

### 专业版

- 无限活跃提醒
- 所有通知方式
- 飞书集成

## 最佳实践

### 1. 合理设置目标价

- 不要设置过于接近当前价格
- 考虑市场波动率
- 参考技术分析位置

### 2. 选择合适的通知方式

- 重要提醒：多种方式
- 普通提醒：单一方式
- 团队协作：飞书通知

### 3. 定期清理

- 删除过期的提醒
- 检查已触发的提醒
- 更新不再有效的目标价

### 4. 结合其他工具

- 与止损止盈策略配合
- 结合技术分析
- 与风险管理系统联动

## 常见问题

### 为什么没有收到通知？

1. 检查提醒是否启用
2. 检查浏览器通知权限
3. 检查价格是否达到目标
4. 检查是否在有效期内

### 循环提醒多久触发一次？

- 价格穿过目标价后触发
- 需要先离开触发区间
- 再次进入时再次触发

**示例**：
```
目标价：$70,000（高于）
当前价：$69,000

1. 价格涨到 $70,001 → 触发
2. 价格继续涨到 $71,000
3. 价格跌回 $69,000
4. 价格再次涨到 $70,001 → 再次触发
```

### 提醒会消耗额度吗？

- 创建提醒不消耗额度
- AI 分析市场需要 AI 额度
- 价格监控完全免费
