## 常见问题

### 账户相关

**Q: 如何注册账户？**
A: 点击页面右上角"注册"按钮，填写邮箱和密码即可注册。

**Q: 忘记密码怎么办？**
A: 在登录页面点击"忘记密码"，通过邮箱重置密码。

**Q: 如何升级订阅计划？**
A: 进入"订阅"页面，选择目标计划完成支付即可升级。

### 功能使用

**Q: 免费版和专业版有什么区别？**
A: 专业版解锁无限回测、AI 助手、高级数据等功能。详见 [订阅计划说明](./subscription.md)。

**Q: AI 助手的使用有限制吗？**
A: 免费版每天 5 条消息，专业版无限制。详见 [AI 助手使用指南](./ai-assistant.md)。

**Q: 如何设置价格提醒？**
A: 在行情页面右侧面板创建价格提醒，详见 [价格提醒配置教程](./alerts.md)。

### 技术问题

**Q: 数据有延迟怎么办？**
A: 刷新页面或检查网络连接。如持续有问题，请联系客服。

**Q: 策略运行出错怎么办？**
A: 查看错误日志，检查策略参数。如无法解决，使用 AI 助手获取帮助。

**Q: API 如何使用？**
A: 访问 [API 文档](/api-docs) 查看详细接口说明和示例代码。
