# 权限和限额说明

AlphaArena 根据订阅计划设置不同的功能限额，帮助用户合理使用平台资源。

## 限额概览

| 功能 | 免费版 | 专业版 | 企业版 |
|------|--------|--------|--------|
| 并发策略 | 3 | ∞ | ∞ |
| 每日回测 | 10 | ∞ | ∞ |
| AI 对话/天 | 5 | ∞ | ∞ |
| 价格提醒 | 5 | ∞ | ∞ |
| 市场数据 | Level 1 | Level 2 | Level 2 |
| API 调用/分钟 | 60 | 300 | 1000 |

## 功能限额详解

### 策略运行

#### 并发策略数

**免费版**：最多同时运行 3 个策略

**专业版**：无限制

**说明**：
- 策略数量包括运行中和暂停的策略
- 删除策略后可释放额度
- 策略失败不影响限额

#### 使用建议

1. **优先级排序**：优先运行表现最好的策略
2. **组合策略**：将相关策略合并为一个
3. **定期清理**：删除不再使用的策略

### 回测功能

#### 每日回测次数

**免费版**：每天 10 次回测

**专业版**：无限制

**说明**：
- 每日 0:00 UTC 重置
- 回测失败不计入次数
- 回测报告保留在历史记录中

#### 使用建议

1. **小范围测试**：先用短时间范围测试
2. **参数优化**：使用网格搜索优化参数
3. **批量回测**：专业版可批量测试多个参数组合

### AI 助手

#### 每日对话次数

**免费版**：每天 5 条消息

**专业版**：无限制

**说明**：
- 包括所有类型的 AI 对话
- 市场分析、策略优化都计入额度
- 每日 0:00 UTC 重置

#### 使用建议

1. **一次性提问**：把多个问题合并为一条消息
2. **精准描述**：提供详细上下文获得更好回答
3. **保存重要回答**：记录有价值的建议

### 价格提醒

#### 活跃提醒数

**免费版**：最多 5 个活跃提醒

**专业版**：无限制

**说明**：
- 只计算"生效中"状态的提醒
- 已触发、已禁用的不计数
- 删除后立即释放额度

#### 使用建议

1. **优先级设置**：为重要价格位设置提醒
2. **及时清理**：删除不再需要的提醒
3. **循环提醒**：一个循环提醒可多次使用

### 市场数据

#### 数据级别

**Level 1（免费版）**：
- 最新成交价
- 买一/卖一价
- 基础成交量

**Level 2（专业版）**：
- 深度订单簿
- 成交明细
- 更多历史数据

#### 使用建议

1. **免费版**：使用基础数据进行简单策略
2. **专业版**：利用深度数据优化交易执行

### API 访问

#### 请求频率限制

| 计划 | 每分钟请求数 |
|------|--------------|
| 免费版 | 60 |
| 专业版 | 300 |
| 企业版 | 1000 |

**说明**：
- 超过限制返回 429 错误
- 等待 1 分钟后自动恢复
- 建议实现客户端限流

## 使用量查看

### 在应用中查看

1. 进入用户仪表板
2. 点击"使用情况"标签
3. 查看各项功能的使用量

### 使用进度条

- 显示当前使用量/限额
- 接近限额时显示警告颜色
- 达到限额时显示红色

## 达到限额时

### 提示信息

当达到限额时，系统会显示：

```
⚠️ [功能名]已达使用限制
您今日已使用 X 次，已达到 X 次的限制。
升级以获取更多使用额度。
```

### 解决方式

1. **等待重置**：每日限额在次日 0:00 UTC 重置
2. **升级计划**：升级到专业版获得无限使用
3. **优化使用**：更高效地使用当前额度

### 升级提示

达到限额时会显示升级按钮：

```
[立即升级]
```

点击跳转到订阅页面。

## 功能锁定

某些功能仅对付费用户开放：

| 功能 | 免费版 | 专业版 |
|------|--------|--------|
| 高级策略 | ❌ | ✅ |
| 风险预警 | ❌ | ✅ |
| 数据导出 | ❌ | ✅ |
| AI 策略优化 | ❌ | ✅ |

### 解锁方式

遇到功能锁定时：

1. 点击功能入口
2. 系统显示升级提示
3. 选择升级计划
4. 完成支付后立即解锁

## 常见问题

### 限额可以购买吗？

目前不支持单独购买限额，请升级到更高计划获得更多额度。

### 限额是固定的吗？

我们可能根据系统负载和用户反馈调整限额，调整前会通知用户。

### 如何查看历史使用量？

目前使用量仅显示当日数据，历史使用量暂不支持查看。

### API 限额可以申请提高吗？

企业版用户可以联系客服申请提高 API 限额。
