# AI 策略助手使用指南

AlphaArena AI 策略助手是一个智能对话系统，可以帮助您分析市场、优化策略、学习交易知识。

## 功能概述

AI 助手可以帮您：

- 📊 **市场分析** - 分析市场趋势、技术指标
- 💡 **策略建议** - 提供策略优化建议
- ⚠️ **风险管理** - 风险评估和管理建议
- 📚 **知识解答** - 解释交易概念和技术指标

## 访问 AI 助手

在页面右下角点击 AI 图标即可打开助手面板：

![AI Assistant Button](./images/ai-button.png)

## 基本使用

### 发送消息

1. 在输入框中输入您的问题
2. 点击发送按钮或按 Enter 键
3. 等待 AI 生成回复

### 快捷操作

AI 助手提供以下快捷按钮：

| 按钮 | 功能 |
|------|------|
| 📊 Market Analysis | 分析指定交易对的市场趋势 |
| 💡 Strategy Tips | 获取策略改进建议 |
| ⚠️ Risk Management | 风险管理建议 |
| 📈 Explain MACD | 解释 MACD 指标 |
| 🎯 Entry Points | 入场点判断建议 |
| 🔄 Exit Strategy | 离场策略建议 |

### 清除对话

点击顶部的删除图标可以清除当前对话历史。

## 使用示例

### 市场分析

```
用户：分析 BTC/USDT 的市场趋势

AI：
BTC/USDT 当前分析：

📊 价格趋势
- 当前价格：$67,234
- 24h 涨跌：+2.3%
- 7日趋势：上涨

📈 技术指标
- RSI (14)：58.3（中性偏多）
- MACD：金叉形成，看涨信号
- MA20：价格在均线上方

💡 建议
- 短期趋势偏多，可考虑逢低布局
- 关注 $66,000 支撑位
```

### 策略优化

```
用户：我的 RSI 策略胜率不高，怎么改进？

AI：
RSI 策略优化建议：

1. 调整超买超卖阈值
   - 传统：70/30
   - 趋势市：80/20
   - 震荡市：65/35

2. 添加确认指标
   - 配合 MACD 背离确认
   - 使用布林带边界确认

3. 优化止损止盈
   - 止损：2-3% ATR
   - 止盈：风险回报比 1:2

4. 过滤假信号
   - 只在趋势方向交易
   - 避免重要数据公布时交易
```

### 风险管理

```
用户：我的投资组合风险如何控制？

AI：
风险管理建议：

📊 当前风险状况
- 总仓位：85%（偏高）
- 集中度：BTC 60%（较集中）

⚠️ 风险提示
1. 建议降低总仓位至 70% 以下
2. 分散持仓，单一资产不超过 40%

🛡️ 风控建议
- 设置总止损：账户净值 -10%
- 单笔止损：不超过账户 2%
- 定期再平衡：每月检查一次
```

## 使用技巧

### 1. 提供上下文

提供更多上下文可以获得更准确的回答：

```
❌ 不好的问法：
"怎么改进策略？"

✅ 好的问法：
"我有一个基于 RSI(14) 的均值回归策略，在 BTC/USDT 4H 周期上运行，
最近 30 天胜率只有 45%，怎么改进？"
```

### 2. 分步骤提问

复杂问题拆分为多个步骤：

```
1. "分析 BTC/USDT 的趋势"
2. "基于当前趋势，适合什么策略？"
3. "这个策略的风险点有哪些？"
```

### 3. 请求具体数值

请求具体数值以便量化评估：

```
"建议的止损止盈比例是多少？"
"建议的仓位大小是账户的百分之几？"
```

## 注意事项

### ⚠️ 投资建议声明

AI 助手提供的分析和建议仅供参考，不构成投资建议。投资有风险，决策需谨慎。

### 🔒 数据隐私

- 对话内容仅用于提供 AI 服务
- 不会与第三方共享您的数据
- 您可以随时删除对话历史

### 📈 实时性

- 市场数据可能有延迟
- 价格信息仅供参考
- 重要决策请核实实时数据
