# 快速入门指南

欢迎使用 AlphaArena！本指南将帮助您在 10 分钟内开始使用平台。

## 注册和登录

### 1. 注册账户

1. 访问 AlphaArena 网站
2. 点击右上角"注册"按钮
3. 填写邮箱和密码
4. 验证邮箱地址

### 2. 首次登录

1. 使用注册的邮箱和密码登录
2. 系统会引导您完成初始设置
3. 设置您的交易偏好

## 开始交易

### 模拟交易

AlphaArena 默认使用模拟交易，无需真实资金：

1. 进入"行情"页面
2. 选择交易对（如 BTC/USDT）
3. 在交易面板输入买卖数量
4. 点击"买入"或"卖出"

### 查看订单

1. 进入"Trades"页面
2. 查看历史订单记录
3. 点击订单查看详情

## 运行策略

### 选择策略

1. 进入"Strategies"页面
2. 浏览可用策略列表
3. 点击策略卡片查看详情

### 启动策略

1. 选择一个策略（如 SMA 交叉策略）
2. 设置策略参数：
   - 交易对：BTC/USDT
   - 时间周期：1H
   - 短周期：10
   - 长周期：20
3. 点击"启动"按钮

### 监控策略

1. 进入"Dashboard"页面
2. 查看策略运行状态
3. 查看实时盈亏

## 使用 AI 助手

### 打开 AI 助手

点击页面右下角的 AI 图标。

### 快速分析

1. 点击"📊 Market Analysis"按钮
2. 选择要分析的交易对
3. 查看 AI 生成的分析报告

### 提问

在输入框中输入问题，例如：

```
BTC/USDT 当前趋势如何？
```

AI 会给出分析结果和建议。

## 设置价格提醒

### 创建提醒

1. 进入"行情"页面
2. 在右侧面板找到"价格提醒"
3. 点击"创建提醒"
4. 填写：
   - 交易对：BTC/USDT
   - 条件：高于目标价
   - 目标价：70000
5. 点击"创建"

### 接收通知

当价格达到目标时，您会收到通知：
- 应用内通知
- 浏览器推送（需授权）

## 查看绩效

### 绩效概览

1. 进入"绩效"页面
2. 查看关键指标：
   - 总收益
   - 夏普比率
   - 最大回撤
   - 胜率

### 详细分析

- 权益曲线：资金变化趋势
- 回撤分析：风险暴露情况
- 收益分布：每日/每周收益

## 回测策略

### 运行回测

1. 进入"回测"页面
2. 选择策略
3. 设置参数：
   - 时间范围
   - 初始资金
   - 手续费率
4. 点击"开始回测"

### 分析结果

- 收益曲线
- 交易记录
- 风险指标
- 参数优化建议

## 订阅计划

### 查看计划

1. 进入"订阅"页面
2. 比较不同计划的功能
3. 选择适合的计划

### 升级

1. 选择计划
2. 选择计费周期（月付/年付）
3. 完成支付

## 新用户提示

### 第一次使用建议

1. **浏览教程**：阅读本指南了解功能
2. **模拟交易**：先用模拟账户熟悉界面
3. **尝试策略**：运行一个简单策略体验
4. **查看绩效**：理解各项指标含义
5. **设置提醒**：配置价格提醒跟踪市场

### 学习路径

```
新手 → 熟悉界面 → 模拟交易 → 运行策略 → 分析绩效 → 优化策略
```

## 常见问题

### 如何重置密码？

1. 点击登录页面的"忘记密码"
2. 输入注册邮箱
3. 按邮件提示重置密码

### 如何切换交易对？

在行情页面的左侧面板选择交易对。

### 如何查看历史数据？

在绩效、交易等页面默认显示历史数据。

### 如何联系客服？

- 使用 AI 助手提问
- 发送邮件至 support@alphaarena.com
